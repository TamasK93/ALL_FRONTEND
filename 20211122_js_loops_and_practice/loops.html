Ciklusok

Olyan feladatokra amiket tobbszor egymas utan kell evegezni.

Okolszabaly:

  • Ha ismerjuk a ciklus ismetlesszamat, akkor for
  • Ha a ciklust egy feltetel vezerli (ismetlesek szama attol fugg, hogy egy feltelte mikor valt igazbol hamissa) akkor while

While

Eloltesztelo ciklus, belepesi feltetel vezerli a ciklus futasat A ciklusmag csak akkor hajtodik vegre ha a feltetel igaz 

while ( feltetel ) { ciklusmag }
while ( condition ) { loop-body }

do while
do {
  // ... loop body ...
} while (condition);

For

for (let ciklisvaltozo = kezdoertek; megallasi feltetel; szamlalo) {
  // ... ciklusmag ...
}
for (begin; condition; step) {
  // ... loop body ...
}

kezdoertek adas csak egyszer fut le a megallasi feltetel minden iteracio elott a szamlalo minden iteracio utan

Feladataok

  • Feladat adjuk ossze az elso 20 szamot. (https://jsbin.com/xuqudev/1/edit?js,console)
  • Output even numbers in the loop [0..100]

Breaking the loop

Eroszakkal lezarja a ciklust

let sum = 0;

while (true) {

  let value = +prompt("Enter a number", '');

  if (!value) break; // (*)

  sum += value;

}
alert( 'Sum: ' + sum );

Continue to the next iteration

Nem olyan eroszakos, mint a break, csak befejezi az aktualis iteraciot es ugrik a kovetkezore 

for (let i = 0; i < 10; i++) {

  // if true, skip the remaining part of the body
  if (i % 2 == 0) continue;

  alert(i); // 1, then 3, 5, 7, 9
}
